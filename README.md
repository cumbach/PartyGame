# PartyGame

## Installing React Native
- `brew install node`
- `brew install watchman`
- `npm install -g react-native-cli`

## Installing the Project
- `git clone https://github.com/cumbach/PartyGame.git`
- `cd PartyGame`
- `npm install`

## Running
- `react-native run-ios`
- `react-native run-android`
